Title - Sub-Palettes
text0001 - The Sub-Palette allows you to add items or Class Skills to it for easy access.<br>One palette can hold a maximum of ten slots.<br><br>You can move the cursor left and right using <%SubPaletteLeft> and <%SubPaletteRight>, <br>and select slots using <%SubPalette>.<br>When using a keyboard, the 1 through 0 keys are mapped to each slot.
